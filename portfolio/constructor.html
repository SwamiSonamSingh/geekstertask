Constructor:-

Constructor is used to perform initillization of object. It is necessary that the name of constructor is same as the class name..

Important Facts:-

 For every object the constructorwill be executed only once.
 Constructor has no return type.
 It allowed public, default, protected, private modifiers.

eg:-

class Student{
String name;
int roll;
Student(String name,int roll){
this.name=name;
this.roll=roll;
}
}
class Test{
public static void main(String[] args){
Student s1=new Student("Ram",10);
System.out.println(s1.name);
System.out.println(s1.roll);
}
}

this & super keyword:-

 this keyword is used to invoke current instance of object
 super keyword is used to invoke parent class object.
 super & this keyword is used anywhere excepr static area.

eg:-

class Parent{
String name="Parent class";
}
class Cild extends Parent{
String name="Child class";
public void m1(){
System.out.println(name);//by default child class
System.out.print(this.name);//current invoking object i.e. child class
System.out.println(super.name);//parent class object
}
} class Test{
public static void main(String[] args){
Child c=new Child();
c.m1(); }
}
