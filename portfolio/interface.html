Interface:-

 Any service requirement specification interface is purely abstract.
 JDBC API & servlet API is default interface.
 In the interface class child class are responsible for implementation and it is compulsoury that the modifier is not removed and also provide implementation af all method. Also, when we don't provide the implementation of all method then we declear class as abstract.

eg:-

Interface I{
public void m1();
public void m2();
}
class Test implements I{
public void m1{
System.out.println("Method m1");
}
public void m2{
System.out.println("Method m2");
}
}
