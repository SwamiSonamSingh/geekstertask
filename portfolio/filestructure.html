Java File Structure:-

A java file contains more than one classes but it is also true that a java file contains at most one public class.

eg:-
class A{
  public static void main(String[] args){
    System.out.println("Class A invoked");
  }
}
class B{
  public static void main(String[] args){
    System.out.println("Class B invoked");
  }
}
class C{
  public static void main(String[] args){
    System.out.println("Class C invoked");
  }
}

 In the above example we can save this file either A.java/B.java/C.java/anyname.java because in this java file there is no any public class. And when the code compiled three class files are created A.class,B.class & C.class and the according to our need we run our desired class file. So when there no public class avilable then we save that java file to any name.

eg:-
class A{
  public static void main(String[] args){
    System.out.println("Class A invoked");
  }
}
public class B{
  public static void main(String[] args){
    System.out.println("Class B invoked");
  }
}
class C{
  public static void main(String[] args){
    System.out.println("Class C invoked");
  }
}

 In the above example we can save this file only as B.java, because it i specify that the java file must be same as the public class.Otherwise it thow us compile time error.
